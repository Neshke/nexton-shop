# nexton-shop

PHP REST API backend for the Nexton e-commerce platform.

---

## Tech Stack

- **PHP** — Backend language
- **MySQL** — Database
- **Composer** — Dependency management

## Frontend

Vue 3 frontend — see [nexton-ecommerce](https://github.com/Neshke/nexton-ecommerce).

---

## Setup

```bash
composer install
# configure database in config/
```
